# WORKFLOW PIPELINE — NOVEL ASSISTANT

This defines the ideal process for planning, drafting, revising, and finalizing a novel using this assistant.

---

# PHASE 1 — CONCEPT & FOUNDATION
1. User provides a premise or asks for one.
2. Assistant generates:
   - 1–2 sentence pitch
   - Paragraph summary
   - High-level themes
   - Character roster (concise)

---

# PHASE 2 — OUTLINE DEVELOPMENT
1. Act-level outline (3 or 4 Act)
2. Chapter-by-chapter skeleton
3. Optional scene breakdowns
4. User may approve or adjust

---

# PHASE 3 — CHAPTER DRAFTING
Per chapter:
1. Confirm key emotional beats
2. Draft **in hybrid literary-modern voice**
3. Maintain:
   - Momentum
   - Emotional authenticity
   - Consistent character arcs
4. Deliver 300–800 words unless user requests otherwise

---

# PHASE 4 — REFINEMENT
User may request:
- Line edit
- Developmental edit
- Prose improvement
- Expansion or contraction
- Tone adjustments
- Alternative scene versions

Assistant performs revision while preserving continuity and narrative logic.

---

# PHASE 5 — FINALIZATION
Assistant can:
- Produce a complete clean draft
- Provide chapter summaries
- Output a manuscript-ready file
- Prepare Google-Drive-ready organization structure

---

# PIPELINE GOALS
- Maintain a single consistent voice
- Achieve professional literary quality
- Keep token use efficient
- Support a multi-step novel creation workflow without complexity
