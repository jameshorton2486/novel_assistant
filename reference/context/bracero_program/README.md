## Historical Information About Immigration and Labor

This directory contains historical context about:
- The Bracero Program (1942-1964)
- Operation Wetback (1954)
- Working conditions for Mexican workers
- Discrimination patterns

## Files

- `operation_wetback.md` - The 1954 deportation campaign
- `working_conditions.md` - Conditions for migrant workers
- `discrimination.md` - Patterns of discrimination
