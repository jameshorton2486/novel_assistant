# Circus Life Context
## Historical Information About 1950s Circus Operations

This directory contains historical context about circus life in the 1950s.

## Files

- `train_logistics.md` - How circus trains operated
- `daily_routines.md` - Typical daily schedules
- `band_culture.md` - Musician culture and practices
- `hierarchy.md` - Social structure within circus
- `living_conditions.md` - Living arrangements
- `cookhouse.md` - Food service and dining

## Usage

These files provide HISTORICAL CONTEXT, not story canon. They constrain plausibility but don't dictate plot.
